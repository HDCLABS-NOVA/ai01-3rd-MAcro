# 🎯 티켓 매크로 (Ticket Macro)

화면 **색상 감지** 기반의 자동 좌석 선택 매크로입니다.  
서버 API/DOM 조작 없이 화면 픽셀 색상을 스캔하여 선택가능한 좌석을 찾아 클릭합니다.

---

## 📁 파일 구조

```
macro/
├── macmain.py       ← 메인 실행 파일
├── macsearcher.py   ← 색상 감지 & 클릭 로직
├── mactray.py       ← 시스템 트레이 아이콘
└── macREADME.md     ← 이 파일
```

> `assets/` 폴더와 이미지 파일은 **더 이상 필요하지 않습니다.**

---

## ⌨️ 단축키

| 키 | 기능 |
|:---:|:---|
| **F2** | 자동 좌석 탐색 — 선택가능 좌석 중 첫 번째(앞줄) 클릭 |
| **F3** | 앞좌석 우선 탐색 — 프리미엄 → 지정석 → 자유석 순서 |
| **F4** | 매크로 종료 |

---

## ⚙️ 내부 동작 원리

```
화면 캡처 (pyautogui)
       ↓
HSV 색상 변환 (OpenCV)
       ↓
좌석 등급별 색상 범위 마스킹
  ├── 🟣 프리미엄 (보라계열)
  ├── 🟢 지정석  (청록계열)
  └── 🟡 자유석  (노란계열)
       ↓
컨투어 감지 → 도트 중심좌표 추출
       ↓
앞줄(위쪽) → 왼쪽 순으로 정렬
       ↓
첫 번째 좌석 자동 클릭 ✅
```

- 미발견 시 0.3초 후 재시도 (최대 50회)
- 클릭 시 ±2px 랜덤 오프셋으로 자연스러운 동작

---

## 🚀 사용 방법

### Step 1. 예매 페이지 준비
예매 사이트에서 **좌석 선택 화면이 보이는 상태**까지 직접 이동합니다.  
(로그인 → 공연 선택 → 날짜 선택 → 캡차 통과까지 수동)

### Step 2. 매크로 실행

```bash
python macro/macmain.py
```

### Step 3. 단축키로 탐색 시작

- **F2** — 화면에서 선택가능 좌석을 찾아 자동 클릭
- **F3** — 프리미엄부터 앞줄 우선 탐색
- 성공 시 터미널에 `✅ 좌석 발견!` 출력

## 🔧 색상 캘리브레이션 (`macsearcher.py`)

실제 화면 색상과 안 맞으면 `SEAT_COLORS`의 HSV 값을 조정하세요.

| 등급 | 색상 | HSV Lower | HSV Upper |
|---|---|---|---|
| 프리미엄 | 보라/핑크 | `[130, 80, 120]` | `[170, 255, 255]` |
| 지정석 | 청록/민트 | `[150, 80, 120]` | `[180, 255, 255]` |
| 자유석 | 노란/주황 | `[15, 100, 150]` | `[35, 255, 255]` |

기타 설정:

| 변수 | 기본값 | 설명 |
|---|---|---|
| `RETRY_INTERVAL` | `0.3` | 재시도 간격 (초) |
| `MIN_DOT_AREA` | `30` | 감지할 도트 최소 픽셀 면적 |
| `MAX_DOT_AREA` | `800` | 감지할 도트 최대 픽셀 면적 |

## ⚠️ 주의사항

- 마우스를 **화면 좌상단 모서리**로 이동하면 긴급 종료됩니다 (`FAILSAFE`)
- 화면 밝기/배율이 달라지면 HSV 값 재조정 필요
- 캡차·대기열은 **수동으로 직접** 처리 후 F2/F3 입력
